// what_this_is

A different kind of hackathon

No pitches. No competition. Just a space to build together with AI, learn from each other, and ship something real.

This is the in-person version of what Learn Vibe Build teaches online — AI as creative partner, building as brainstorming, articulation as the core skill.

🤝

Collaborative

Demo sessions, design feedback, round tables. Help each other's projects. Fluid contribution across ideas.

🧠

Bonfires Knowledge Graph

Live collective intelligence. Query what's being discussed. Build on shared context.

🌱

Open Source

Everything created is forkable, remixable, owned by the commons.
