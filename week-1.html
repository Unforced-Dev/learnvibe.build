// the_landscape

What is an LLM?

Pattern matching on human knowledge at scale.
Trained on billions of documents, conversations, code.

Good at: language, reasoning, code generation, creative tasks

The chat interface is just the beginning.
It's the simplest way to interact, but not the most powerful.

// the_landscape

What is an Agent?

LLM + Tools + Loop = Agent

→ Tools: ability to take actions (read files, browse web, run code)

→ Loop: can iterate until the task is done

→ Autonomy: makes decisions, recovers from errors

// thinking_with_self

Thinking with Self

→ Self-knowledge: Know what you want to build. Know your limitations.

→ Know what you don't know: Easier to bring into context, use tech or relationships to infill.

→ Felt sense: Your intuition about direction matters. Being in touch with your body is a skill.

In the world of vibecoding, your felt sense of what direction to go
is actually a crucial component.

// thinking_with_ai

Thinking with AI

→ Not just building — inquiring. The thinking process is key.

→ Share what you're thinking. Ask for other perspectives.

→ Express uncertainty. "I want to think before building."

→ Get interviewed. Ask AI to ask you questions to find the root of what you're building.

// ai_inquiry

Try saying things like...

"I have an idea but I'm not sure about the shape of it yet. Can you ask me questions to help me clarify?"

"Before we build anything, I want to think through this with you. What are the key decisions I need to make?"

"Interview me about this project. Help me discover what I actually want."

// thinking_with_others
