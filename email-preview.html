Optional but helpful: Create a GitHub account before Tuesday

We'll be talking about GitHub briefly, and having an account will let you follow along. It's free and only takes a couple minutes:

  1. Go to https://github.com
  2. Click "Sign up"
  3. Use your email, create a username and password
  4. That's it — you don't need to create any repositories yet

GitHub is where code lives on the internet. Having an account means you'll eventually be able to save your projects, collaborate with others, and even host websites for free.
