MCPs / Connectors

MCP = Model Context Protocol
In Claude's UI: Connectors or Integrations

Ways for Claude to reach beyond the conversation and interact with your world.

// mcps

What MCPs enable

→ Search the web for current information

→ Access your files — Google Drive, Notion, etc.

→ Connect to services — databases, APIs

→ Read/write files on your computer

Think of it as giving Claude "hands" to reach out and touch things.

→ Organize files — folder structures, move things around

Cowork vs Claude Code

→ Same capabilities — both read/write files, organize, build

→ Different interface — Cowork is visual, Code is terminal

→ Code goes further — run commands, install, deploy

→ Platform — Code works on Mac, Windows, Linux

// github

GitHub Basics

Where code lives on the internet.

→ Storage — like Google Drive for software

→ Version control — tracks every change

→ Collaboration — multiple people, same project

→ Hosting — serve websites directly

// github

Public vs Private

Public

  • • Anyone can see
  • • Open source
  • • Free website hosting
  • • Collaboration

Private

  • • Only you can see
  • • Client work
  • • Personal projects
  • • Free (with limits)

Never commit:

API keys, passwords, .env files, personal data
