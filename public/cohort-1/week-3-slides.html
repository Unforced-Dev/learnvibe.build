// paradox

The faster the tools get,

// paradox

Why slow down?

→ Speed creates urgency — pressure to add more, faster

→ Complexity compounds — each feature adds surface area

→ Understanding matters — if you can't explain it, slow down

The AI doesn't feel complexity. You will when things break.

// paradox

Practices for staying grounded

→ Take breaks — step away, come back with fresh eyes

→ Write first, build second — articulate before asking

→ Review what was built — understand what Claude created

→ Trust your felt sense — if it feels off, pause

// git

Git & GitHub

Version control.
Where code lives.

// git

What is Git?

→ Go back in time — undo mistakes, restore versions

→ Branch — try experiments safely

→ Merge — bring changes together

→ Collaborate — multiple people, same project

Git tracks every change. It's your safety net.

// git

Key concepts

Repository

A folder Git tracks. Your code + its history.

Commit

A snapshot. Like a save point in a game.

Branch

A parallel version. Try things without breaking main.

Worktree

Multiple branches at once. Claude Code uses this.

// github

What is GitHub?

→ Collaboration hub — share, get contributions

→ Free hosting — GitHub Pages serves websites

→ World's code library — millions of projects
